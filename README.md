# Для глаз

Лёгкое нативное приложение в menu bar macOS: напоминает делать перерывы для глаз
(пошаговый 20-секундный guided-flow) и вставать на разминку (умное напоминание,
которое понимает по активности, вставал ты или нет).

## Возможности
- **Глаза** каждые ~20 мин: карточка по центру с кольцом-отсчётом 20 сек и шагами
  «моргание → дальний фокус → расслабление». Закрывается сама в 0 или по Esc/крестику.
- **Разминка** каждые ~60 мин: «Встань и пройдись 3–5 минут».
  - «Встал / иду» (или кнопка **«Встаю на разминку»** в меню) — переводит в режим
    разминки: таймеры замирают. Разминка засчитывается, **только** когда ты реально
    отошёл (нет активности / экран заблокирован / сон) и вернулся, пробыв вне компа
    достаточно долго, — либо вручную кнопкой **«Сел работать»**.
  - «Ещё работаю» — прячет карточку, но счётчик **«Не вставал: N мин»** растёт в меню.
  - Отдых засчитывается автоматически только при **подтверждённом** отсутствии:
    блокировка экрана или сон Mac (по длительности). Просто «нет ввода» (читаешь
    статью / смотришь видео не в fullscreen) ставит таймеры на **паузу, но не сбрасывает** —
    ты ведь всё это время смотрел в экран.
- **Умная пауза:** когда отошёл от компа (idle/экран заблокирован) или в полноэкранном
  режиме — таймеры замирают.
- **Адаптивно** к 13"/15" MacBook; в настройках есть размер карточек S/M/L.
- Автозапуск при входе, звук уведомления, тестовый режим (быстрые таймеры 10с/20с).

## Сборка
```bash
./build_app.sh
```
На выходе — `Для глаз.app` рядом со скриптом.

## Запуск / установка
```bash
open "Для глаз.app"
# или в Программы:
cp -R "Для глаз.app" /Applications/
xattr -dr com.apple.quarantine "/Applications/Для глаз.app"
open "/Applications/Для глаз.app"
```
Приложение без иконки в Dock — управление через значок в строке меню (человечек в кружке).

## Быстрая проверка
Меню → **Настройки → Основное → Тестовый режим** (таймеры 10с / 20с), либо кнопки
**Тест: глаза** / **Тест: встать** в меню.

## Тесты
```bash
swift test
```
Юнит-тесты на логику scheduler (swift-testing): fullscreen без ввода ≠ уход, блокировка/сон
засчитываются как отдых, snooze после «Ещё работаю», отложенная разминка после глазного перерыва.
Время и активность инжектируются (`ActivityProviding` / `FullscreenProviding` / `now`), поэтому
тесты детерминированы. В тестовом режиме (Настройки → Тестовый режим) в меню показывается debug-строка.

## Технологии
Swift 6 + SwiftUI + AppKit, без сторонних зависимостей. Idle через IOKit `HIDIdleTime`
(без запроса прав), автозапуск через `SMAppService`.
